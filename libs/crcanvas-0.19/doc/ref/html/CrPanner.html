Property Details

The "active" property

  "active"                   gboolean              : Read / Write

Active/Deactivate the panner object or check the activation status.

Default value: FALSE

The "button" property

  "button"                   gint                  : Read / Write

Mouse button implicitly activates the panner.

Allowed values: [0,10]

Default value: 2

The "canvas" property

  "canvas"                   CrCanvas*             : Read / Write

Reference to CrCanvas widget.

The "cursor" property

  "cursor"                   gint                  : Read / Write

GDK Cursor to use when panner is selected.

Allowed values: [0,153]

Default value: 60

The "item" property

  "item"                     CrItem*               : Read / Write

Reference to CrItem from which to collect the button events. Usually this is the canvas root item.
